Download and install Java SDK from:

oracle.com
Java SE Development Kit 8u371

The steps for installing Java SDK:

  1. Downloads -> Java SE -> version 8xxx
  2. Install
  3. Add the JDK path to system PATH environment variable
  4. Test the installation with command

Check that java has the correct version and it is saved in path:

  1. Open CMP
  2. Enter the keywords: "java -version"
